aacgmeval

Syntax
aacgmeval --help
aacgmeval [-i] [-lat lat] [-lon lon] [-alt alt] [-fmt format]
aacgmeval [-i] [-f filename] [-fmt format]
Synopsis

Convert to and from Altitude Adjusted Corrected Geo-Magnetic Coordinates v2 (AACGM_v2).

Options
--helpprint the help message and exit.
--versionprint the RST version number and exit.
-iperform the inverse conversion from AACGM to geographic.
-lat latcalculate for the latitude lat.
-lon loncalculate for the longitude lon.
-alt altcalculate for the altitude in kilometers, alt.
-d yyyymmddcalculate for the date yyyymmdd (v2 only).
-t hr:mncalculate for the time hr:mn (v2 only).
-fmt formatuse the string fmt, to format the output.
-f filenameread a sequence of latitudes, longitudes and altitudes from the ASCII file, filename.
-old_aacgmuse old AACGM coordinates rather than v2.
Description

Converts to and from Altitude Adjusted Corrected Geo-Magnetic Coordinates v2 (AACGM_v2) [Shepherd 2014].The output is written to standard output using the formatting string given by the "-fmt" option. The format string should be in the same format as that used by the printf function and will take two double precision floating point numbers as arguments.

If the "-f" option is given, the latitude,longitude and altitude are read from a space separated ASCII file. If the filename is "-" input is read from standard input.

Examples
aacgmeval -lat 85.4 -lon 32.4 -alt 300.0

Convert the coordinate given by latitude of 85.4 degrees, longitude 32.4 degrees and altitude of 300 kilometers to AACGM.

aacgmeval -i -f aacgm.txt

Convert the AACGM coordinates in the file "aacgm.txt", back to geographic coordinates.
