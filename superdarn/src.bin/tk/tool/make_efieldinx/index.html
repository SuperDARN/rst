make_efieldinx

Syntax
make_efieldinx --help
make_efieldinx [efieldname]
Synopsis

Creates an index of a electric field data.

Options
--helpprint the help message and exit.
--versionprint the RST version number and exit.
efieldnamefilename of the electric field data. If this is omitted the file is read from standard input.
Description

Creates an index of eletric field data.

The index is written to standard output.

Examples
make_efieldinx 20040830.efield > 20040830.efieldinx

Create an index file for the electric field data file "20040830.efield". The index is writen to the file "20040830.efieldinx".
