ConvertBitOrder

Syntax
int ConvertBitOrder();
Header
base/rconvert.h
Library
rcnv
Description

The ConvertBitOrder function determines whether the system is little-endian or big-endian.

Returns
Returns zero if the system has little-endian architecture, or a non zero value if it is big-endian.
Errors
None
