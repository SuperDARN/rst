AACGMInit

Syntax
int AACGMInit(int yr);
Header
analysis/aacgm.h
Library
aacgm
Description

The AACGMInit will load the coefficient set for the year given by the argument yr.

The prefix of the coefficient filename should be given by the environment variable AACGM_DAT_PREFIX, the four digit year is appended to this to create the full filename.

Returns
Returns zero on success. On error, a (-1) is returned.
Errors
On error, a (-1) is returned.
Example

Source Code: AACGMInit.c

/* AACGMInit.c
   ===========
   Author: R.J.Barnes
*/


#include <stdio.h>
#include <stdlib.h>
#include "aacgm.h"


int main(int argc,char *argv[]) {

  double ilat,ilon,hgt;
  double olat,olon,r;
  int yr;
  int flg;
  int s;

  ilat=85.0;
  ilon=45.0;
  hgt=150.0;

  flg=0;

  yr=1990;
  s=AACGMInit(yr);
  if (s !=0) {  
    fprintf(stderr,"Error in initializing AACGM'n");
    exit(-1);
  }
  s=AACGMConvert(ilat,ilon,hgt,&olat,&olon,&r,flg);
  if (s !=0) {
    fprintf(stderr,"Error in AACGM conversion.'n");
    exit(-1);
  }
  fprintf(stdout,"AACGMInit'n");
  fprintf(stdout,"AACGM Year:%d'n",yr);
  fprintf(stdout,"Input: lat=%g, lon=%g, height=%g'n",ilat,ilon,hgt);
  fprintf(stdout,"Output: lat=%g, lon=%g'n",olat,olon);
  return 0;
}
