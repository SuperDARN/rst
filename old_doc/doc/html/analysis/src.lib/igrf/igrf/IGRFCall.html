IGRFCall

Syntax
int IGRFCall(double date,double lat,double lon,double elv,double *x,double *y,double *z);
Header
analysis/igrfcall.h
Library
igrf
Description

The IGRFCall function calculates the components of the geomagnetic field according to the International Geomagnetic Reference Field (IGRF) model.

The argument date is the decimal date used to calculate the field. The arguments lat, lon, and elv give the latitude and longitude in degrees, and the the height in kilometers at which to calculate the field. The components of the field are stored at the locations pointed to by the arguments x, y, and z in local north/east/down coordinates and units of nT.

Returns
Returns zero on success. On error, a (-1) is returned.
Errors
On error, a (-1) is returned.
Example

Source Code: IGRFCall.c

/* igrfcall.c
   ========== 
   Author: R.J.Barnes
*/

#include <stdio.h>
#include <stdlib.h>
#include "igrfcall.h"

int main() {
   double dte=1997.0;
   double lat=83.0;
   double lon=110.0;
   double elv=100.0;

   double x,y,z;
 
   fprintf(stdout,"lat=%g lon=%g elv=%g'n",lat,lon,elv);
 
   IGRFCall(dte,lat,lon,elv,&x,&y,&z);

   fprintf(stdout,"x=%g y=%g z=%g'n",x,y,z);
   return 0;
}
