Description

The struct RfileData structure stores a single data value from an rFile format file.

Possible values of the type field are:

FIELD_CHAR
A single character.
FIELD_INT
A signed integer.
FIELD_DOUBLE
A double precision floating point number.
FIELD_TXT
A text string.
