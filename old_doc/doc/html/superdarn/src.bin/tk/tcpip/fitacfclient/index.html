fitacfclient

Syntax
fitacfclient --help
fitacfclient [-s stream] host port
Synopsis

Simple client program for fitacf TCP/IP data streams.

Options
--helpprint the help message and exit.
--versionprint the RST version number and exit.
-s streamfor a multiplexed stream, read the stream number given by stream. A value of 255 will read all streams
hosthostname or IP address of the system to connect to.
portport number to connect to on the system.
Description

Simple client program for fitacf TCP/IP data streams.

The program dumps an ASCII representation of the fitacf data stream to standard output.

If the stream from the server is multiplexed then the "-s" option can be used to select the appropriate stream.

Examples
fitacfclient peanut.jhuapl.edu 1024

Connect to the host peanut.jhuapl.edu at port 1024 and display the data stream.

fitacfclient -s 2 peanut.jhuapl.edu 1024

Connect to the host peanut.jhuapl.edu at port 1024 and display data stream number 2.
